            <section id="o-que-e-lixo-eletronico">
                <h2>O que é lixo eletrônico?</h2>
                <p>Lixo eletrônico, também conhecido como e-lixo ou Resíduo de Equipamento Elétrico e Eletrônico (REEE), é todo dispositivo que depende de corrente elétrica ou campos eletromagnéticos para funcionar e que foi descartado por seu proprietário. Isso inclui desde grandes eletrodomésticos até pequenos dispositivos e seus componentes. Celulares, computadores, televisores, impressoras, geladeiras, cabos, carregadores, baterias, placas-mãe, chips e controles remotos são exemplos comuns, mesmo que não funcionem mais.</p>
            </section>

            <section id="por-que-descartar-corretamente">
                <h2>Por que descartar corretamente?</h2>
                <p>O descarte inadequado de lixo eletrônico representa sérios riscos. Muitos equipamentos contêm metais pesados (como chumbo, mercúrio, cádmio) e outras substâncias tóxicas que, se liberadas no meio ambiente, podem contaminar o solo, a água subterrânea e até o ar (em caso de queima). Essa contaminação afeta ecossistemas e pode causar graves problemas de saúde pública.</p>
                <p>Por outro lado, o descarte correto permite a reciclagem de materiais valiosos presentes nos eletrônicos, como ouro, prata, cobre e plásticos. Isso reduz a necessidade de extrair novos recursos naturais, economiza energia e diminui a emissão de gases de efeito estufa. Em Belo Horizonte, como em todo o Brasil, esse cuidado está alinhado com a Política Nacional de Resíduos Sólidos (Lei nº 12.305/2010) e a Política Nacional do Meio Ambiente (Lei nº 6.938/81), que estabelecem responsabilidades compartilhadas para a gestão adequada desses resíduos.</p>
            </section>

                    <div class="accordion-content" id="content-pilhas" role="region" aria-labelledby="header-pilhas" style="display: none;">
                        <ul>
                            <li><strong>Riscos específicos:</strong> Pilhas e baterias contêm produtos químicos corrosivos e metais pesados. Vazamentos podem contaminar o ambiente e são perigosos ao toque.</li>
                            <li><strong>Separe do lixo comum:</strong> Nunca descarte pilhas e baterias no lixo doméstico indiferenciado.</li>
                            <li><strong>Encontre coletores específicos ("Papa-Pilhas"):</strong> Muitos supermercados, farmácias e lojas de eletrônicos possuem coletores dedicados. Alguns <a href="pontos.html">Pontos de Descarte</a> também os aceitam.</li>
                            <li><strong>Baterias de eletrônicos (celulares, notebooks):</strong> Se forem removíveis, trate-as como outras baterias. Caso contrário, o aparelho completo deve ser levado a um ponto de coleta de eletrônicos. Não tente remover baterias não projetadas para substituição pelo usuário, pois isso pode ser perigoso.</li>
                            <li><strong>Proteja os polos:</strong> Para baterias de lítio (comuns em notebooks, câmeras), é uma boa prática isolar os contatos/polos com fita isolante antes do descarte para evitar curtos-circuitos.</li>
                        </ul>

// Função para exibir o modal de ALERTA personalizado
function showCustomAlert(message) {
  return new Promise((resolve) => {
    const overlay = document.getElementById('customAlertOverlay');
    const messageEl = document.getElementById('customAlertMessage');
    const okButton = document.getElementById('customAlertOkButton');

    messageEl.textContent = message;
    overlay.style.display = 'flex';

    const newOkButton = okButton.cloneNode(true);
    okButton.parentNode.replaceChild(newOkButton, okButton);

    let escapeListener;

    const handleClose = () => {
      overlay.style.display = 'none';
      document.removeEventListener('keydown', escapeListener);
      resolve(true);
    };

    newOkButton.addEventListener('click', handleClose);

    escapeListener = (event) => {
      if (event.key === 'Escape' && overlay.style.display === 'flex') {
        handleClose();
      }
    };
    document.addEventListener('keydown', escapeListener);
  });
}

// Função para exibir o modal de CONFIRMAÇÃO personalizado
function showCustomConfirm(message) {
  return new Promise((resolve) => {
    const overlay = document.getElementById('customConfirmOverlay');
    const messageEl = document.getElementById('customConfirmMessage');
    const okButton = document.getElementById('customConfirmOkButton');
    const cancelButton = document.getElementById('customConfirmCancelButton');

    messageEl.textContent = message;
    overlay.style.display = 'flex';

    const newOkButton = okButton.cloneNode(true);
    okButton.parentNode.replaceChild(newOkButton, okButton);
    const newCancelButton = cancelButton.cloneNode(true);
    cancelButton.parentNode.replaceChild(newCancelButton, cancelButton);

    let escapeListener;

    const handleResolve = (value) => {
      overlay.style.display = 'none';
      document.removeEventListener('keydown', escapeListener);
      resolve(value);
    };

    newOkButton.addEventListener('click', () => handleResolve(true));
    newCancelButton.addEventListener('click', () => handleResolve(false));

    escapeListener = (event) => {
      if (event.key === 'Escape' && overlay.style.display === 'flex') {
        handleResolve(false);
      }
    };
    document.addEventListener('keydown', escapeListener);
  });
}

document.addEventListener('DOMContentLoaded', function () {
    const userIcon = document.getElementById('userIcon');
    const userDropdown = document.getElementById('userDropdown');

    if (userIcon && userDropdown) {
        userIcon.title = "Opções do Usuário";
        userIcon.setAttribute('aria-label', 'Opções do Usuário');

        userIcon.addEventListener('click', function(event) {
            userDropdown.classList.toggle('active');
            event.stopPropagation();
        });

        document.addEventListener('click', function(event) {
            if (!userDropdown.contains(event.target) && !userIcon.contains(event.target) && userDropdown.classList.contains('active')) {
                userDropdown.classList.remove('active');
            }
        });

        const logoutButtonFromDropdown = userDropdown.querySelector('.logout-button');
        if (logoutButtonFromDropdown) {
            logoutButtonFromDropdown.addEventListener('click', async function(event) {
                event.preventDefault();
                const userChoice = await showCustomConfirm("Você tem certeza que deseja sair?");
                if (userChoice) {
                    localStorage.removeItem('usuarioLogadoEcoTechBH');
                    await showCustomAlert('Redirecionando para login...');
                    window.location.href = 'login.html';
                }
            });
        }
    }

    const menuItems = document.querySelectorAll('.menu-list a');
    const currentPath = window.location.pathname.split('/').pop();

    menuItems.forEach(item => {
        const itemHref = item.getAttribute('href');
        if (itemHref === currentPath) {
            item.classList.add('active');
        } else if (currentPath === '' && itemHref === 'index.html') {
            item.classList.add('active');
        } else {
            item.classList.remove('active');
        }
    });

    const accordionHeaders = document.querySelectorAll('.accordion-header');
    accordionHeaders.forEach(header => {
        header.addEventListener('click', function () {
            toggleAccordionItem(this);
        });

        header.addEventListener('keydown', function (event) {
            if (event.key === 'Enter' || event.key === ' ') {
                event.preventDefault();
                toggleAccordionItem(this);
            }
        });
    });

    function toggleAccordionItem(headerElement) {
        const contentId = headerElement.getAttribute('aria-controls');
        const content = document.getElementById(contentId);
        const isExpanded = headerElement.getAttribute('aria-expanded') === 'true';

        headerElement.setAttribute('aria-expanded', !isExpanded);
        if (isExpanded) {
            content.style.display = 'none';
        } else {
            content.style.display = 'block';
        }
    }

    document.addEventListener('keydown', function(event) {
        if (event.key === 'Escape') {
            if (userDropdown && userDropdown.classList.contains('active')) {
                userDropdown.classList.remove('active');
            }
        }
    });
});
